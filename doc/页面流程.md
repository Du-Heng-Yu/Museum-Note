# 页面流程文档

## 1. 整体导航结构

```
App
├── 顶部标题栏（AppHeader，全局显示于 Tab 页面）
│   ├── 左侧：大字"博物馆笔记" + 小字"museum notes"
│   └── 右侧：🔍 搜索按钮（预留）+ 👤 用户头像（预留）
│
├── BottomTabNavigator（底部 Tab 导航）
│   ├── Tab 1：时间轴（TimelineTab）
│   └── Tab 2：我的展览（ExhibitionsTab）
│       [FAB "+" 悬浮在两个 Tab 中间]
│
├── Stack：文物详情页（ArtifactDetail）
├── Stack：文物编辑页（ArtifactEdit）
├── Stack：展览详情页（ExhibitionDetail）
└── Stack：展览编辑页（ExhibitionEdit）
```

---

## 2. 底部导航栏

- **2 个 Tab**：时间轴 / 我的展览
- **全局 FAB**（悬浮操作按钮）位于两个 Tab 图标中间，固定在底部，始终可见
- FAB 样式：圆形"+"按钮，视觉上突出于导航栏

---

## 3. 页面详情

### 3.1 时间轴页（Timeline）

**功能：**
- 纵向滚动列表，每个朝代渲染为一张带颜色的圆角卡片
- 卡片颜色按预定义的颜色序列循环分配（参见 `doc/朝代卡片颜色.png`）
- 排列顺序：顶部为最近的时代（中华人民共和国），依次向下直至旧石器时代
- 朝代数据来自 `dynasties.ts` 常量

**朝代卡片 — 默认（收起）状态：**
- 卡片左侧：大字体显示朝代名称，其下方小字体显示起止年份
- 年份格式：公元后年份正常显示（如 `1644-1912`）；公元前年份以 "BC" 前缀替代负号（如 `-1046` 显示为 `BC1046`，跨公元显示为 `BC206-220`）
- 卡片右侧：横向排列该朝代下的文物缩略图预览（正方形照片 + 下方文物名称小字），缩略图仅用于展示，**不可点击**
- 无照片的文物使用预置占位图

**朝代卡片 — 展开状态：**
- 用户点击默认状态的朝代卡片触发展开（参见 `doc/朝代卡片点击展开后UI.png`）
- 展开动画：卡片下边缘向下延伸（高度增大），过渡丝滑（使用 LayoutAnimation 或 Animated）
- 展开后，右侧的缩略图预览消失，卡片内部在朝代名称 + 年份的下方渲染完整的文物卡片网格：
  - 每排 3 个文物卡片，向下排列
  - 文物卡片：圆角矩形，内部顶部为正方形照片缩略图（文物第一张照片），底部小字显示文物名称
  - 展开状态下的文物卡片**可以点击**，点击跳转至文物详情页

**互斥展开规则：**
- 同一时间只允许一个朝代处于展开状态
- 点击朝代 A 展开后，再点击朝代 B，朝代 B 展开同时朝代 A 自动收起
- 再次点击已展开的朝代可收起

**点击交互：**
- 展开状态下，点击文物卡片 → 跳转至文物详情页

**空状态：**
- 未录入任何文物时，显示引导提示："点击下方 + 开始记录第一件文物"

---

### 3.2 我的展览页（Exhibitions）

**功能：**
- 列表展示所有已创建的展览
- 每个展览卡片显示：展览名称、博物馆名、参观日期、文物数量
- 右上角"+ 添加展览"按钮 → 跳转至展览编辑页（新建模式）

**点击交互：**
- 点击展览卡片 → 跳转至展览详情页

**空状态：**
- 未创建任何展览时，显示引导提示："还没有展览记录，可点击右上角 + 新建展览，或点击下方 + 录入第一件文物"

---

### 3.3 FAB 点击流程（添加文物）

点击底部 FAB "+" 后，底部弹出操作菜单（Bottom Sheet），包含三个选项：

#### 选项 A：从相册选择
```
点击"从相册选择"
→ 调用系统相册（expo-image-picker）
→ 用户多选照片，点击确认
→ 获取临时路径（cache 目录），暂存于页面状态
→ 跳转至文物编辑页（ArtifactEdit）
  → photos 字段预填为临时路径（预览用）
  → 光标自动 focus 到"文物名称"输入框，键盘自动弹出
  → 保存时照片从临时路径复制到 App 私有目录（documentDirectory/photos/）
```

#### 选项 B：相机拍摄
```
点击"相机"
→ 调用系统相机（expo-image-picker 的 launchCameraAsync）
→ 用户拍摄一张照片，点击确认
→ 获取临时路径，同时通过 expo-media-library 保存一份到系统相册
→ 跳转至文物编辑页（ArtifactEdit）
  → photos 字段预填为临时路径（预览用）
  → 光标自动 focus 到"文物名称"输入框，键盘自动弹出
  → 保存时照片从临时路径复制到 App 私有目录（documentDirectory/photos/）
```

#### 选项 C：文字记录
```
点击"文字记录"
→ 直接跳转至文物编辑页（ArtifactEdit）
  → photos 字段为空
  → 光标自动 focus 到"文物名称"输入框，键盘自动弹出
```

### 3.4 文物编辑页（ArtifactEdit）

**进入来源：**
- FAB → 从相册选择 / 相机 / 文字记录（新建模式）
- 文物详情页 → 点击"编辑"（编辑模式）

**字段：**

| 字段 | 交互方式 | 必填 |
|------|----------|------|
| 文物名称 | 文本输入框，进入页面自动 focus | ✅ |
| 年份与朝代模式 | 互斥单选：「自动选择朝代」（默认）/ 「自动选择年份」；App 持久化记住用户上次选择 | ✅ |
| 年份 | 数字输入框；「自动选择朝代」模式下可编辑；「自动选择年份」模式下置灰禁用 | ✅ |
| 中国史朝代 | 下拉选择框（数据来自 dynasties.ts）；「自动选择年份」模式下可选；「自动选择朝代」模式下置灰禁用 | ✅ |
| 所属展览 | 下拉选择列表（拉取 exhibitions 表），底部有"+ 新建展览"入口 | ✅ |
| 照片 | 图片预览区，支持继续添加或删除 | 选填 |
| 文物说明 | 多行文本输入框 | 选填 |
| 个人备注 | 多行文本输入框 | 选填 |
| 标签 | 标签输入，支持多个；输入框提示示例：青铜器、瓷器、玉器 | 选填 |

**年份与朝代联动规则（互斥模式）：**
- **「自动选择朝代」（默认）**：年份输入框可编辑，朝代下拉框禁用；用户输入年份，朝代实时匹配 `dynasties.ts` 并联动更新
- **「自动选择年份」**：朝代下拉框可选，年份输入框置灰禁用；用户选择朝代，年份自动填入该朝代起始年份
- App 持久化记住用户最后一次的模式选择，下次进入编辑页时恢复

**所属展览-现场新建流程：**
```
点击"+ 新建展览"
→ 跳转至展览编辑页（ExhibitionEdit，新建模式）
→ 保存后自动选中该展览，回到文物编辑页
```

**保存：**
- 右上角"保存"按钮，必填字段验证通过后：
  1. 将照片从临时路径复制到 `documentDirectory/photos/`（命名：`artifact_{id}_{index}.jpg`）
  2. 数据库 `photos` 字段写入复制后的私有路径
  3. 其余字段写入数据库
- 保存成功后返回上一页

---

### 3.5 文物详情页（ArtifactDetail）

**进入来源：**
- 时间轴页 → 点击文物卡片
- 展览详情页 → 点击文物卡片

**内容：**
- 照片横向滚动查看（无照片则不显示）
- 文物名称、年份、朝代
- 所属展览：显示"展览名称 \ 博物馆名称"（博物馆信息来自 exhibitions 表），可点击跳转至展览详情页
- 文物说明、个人备注、标签

**操作：**
- 右上角"编辑"按钮 → 跳转至文物编辑页（编辑模式）
- 右上角"删除"按钮 → 弹出确认对话框，确认后删除文物记录并同步删除 `documentDirectory/photos/` 中对应的照片文件，返回上一页

---

### 3.6 展览详情页（ExhibitionDetail）

**进入来源：**
- 我的展览页 → 点击展览卡片
- 文物详情页 → 点击所属展览名称

**内容：**
- 展览名称、博物馆、展览说明、参观日期
- 该展览下所有文物的卡片列表

**点击交互：**
- 点击文物卡片 → 跳转至文物详情页
- 右上角"编辑" → 跳转至展览编辑页（ExhibitionEdit，编辑模式）

---

### 3.7 展览编辑页（ExhibitionEdit）

**进入来源：**
- 我的展览页 → 右上角"+ 添加展览"（新建模式）
- 文物编辑页 → 所属展览选择中点击"+ 新建展览"（新建模式）
- 展览详情页 → 右上角"编辑"（编辑模式）

**字段：**

| 字段 | 必填 | 备注 |
|------|------|------|
| 展览名称 | ✅ | |
| 博物馆 | ✅ | |
| 展览说明 | 选填 | |
| 参观日期（日期选择器） | ✅ | 默认为系统当前日期，可选过去或未来任意日期，无范围限制 |

**保存：**
- 右上角"保存"按钮，必填字段验证通过后写入 exhibitions 表
- 新建模式 + 来自文物编辑页：保存后自动选中该展览，返回文物编辑页
- 新建模式 + 来自我的展览页：保存后返回我的展览页
- 编辑模式：保存后返回展览详情页

**删除（仅编辑模式显示）：**
- 底部"删除展览"按钮 → 弹出二次确认对话框，提示："删除后该展览下的所有文物也将一并删除，此操作不可撤销"
- 确认后级联删除展览及其下所有文物，同步删除对应文物的照片文件，返回我的展览页

---

## 4. 页面跳转关系图

```
BottomTab
├── 时间轴页
│   └── 点击朝代卡片 → 展开朝代
│       └── 点击文物卡片 → 文物详情页
│                            ├── 点击编辑 → 文物编辑页
│                            └── 点击展览名称 → 展览详情页
│                                                └── 右上角编辑 → 展览编辑页（编辑模式）
│
├── 我的展览页
│   ├── 右上角+ → 展览编辑页（新建模式）
│   └── 点击展览 → 展览详情页
│                   ├── 点击文物 → 文物详情页
│                   │               ├── 点击编辑 → 文物编辑页
│                   │               └── 点击展览名称 → 展览详情页
│                   └── 右上角编辑 → 展览编辑页（编辑模式）
│
└── FAB +
    ├── 从相册选择 → [系统相册] → 文物编辑页
    │                               └── 选展览时+ → 展览编辑页（新建模式，返回后继续编辑）
    ├── 相机 → [系统相机] → 文物编辑页
    │                         └── 选展览时+ → 展览编辑页（新建模式，返回后继续编辑）
    └── 文字记录 → 文物编辑页
                    └── 选展览时+ → 展览编辑页（新建模式，返回后继续编辑）
```
